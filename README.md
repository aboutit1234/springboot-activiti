#springboot-activiti

###项目本地部署###
1. 进入项目根目录，执行`gradle build` 安装`spring boot`依赖
2. 安装`js`依赖: `npm install`
3. 执行`bower install` 安装`angular js`依赖
5. 打开 http://localhost:9001 进行访问

### 项目介绍
1. 后台框架基于Spring Boot，Spring Security
2. 工作流使用Activiti
3. 前台框架基于Angular js
